# Tier Ranking App

This project is a simple full-stack web application built with React Router v7 and an Express backend. Users can rank items within various categories using drag-and-drop tiers (S through D) and view aggregated rankings from all submissions.

## Development

- `npm run dev` – start the Vite development server.
- `npm start` – run the Express server (serves built files from `dist`).
- `npm run build` – build the frontend for production.

## Testing

Run unit tests with:

```bash
npm test
```

This executes `uvx pytest`.
