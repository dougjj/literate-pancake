# Tier Ranking App

This project uses React Router v7 in framework mode to allow users to create tier rankings for various categories.

## Features
- Home page with links to ranking categories
- Drag-and-drop tier list for each category
- Submit rankings to update aggregated results stored on the server
- View aggregated tier rankings combining votes from all users

## Development
- `npm run dev` – start the development server
- `npm run build` – build client and server bundles

## Testing
Run the tests (none provided) with:

```
uvx pytest
```
